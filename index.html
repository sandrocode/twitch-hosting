How to easily host a channel on Twitch?

Why streamers ask to you to host their channel?

Because hosting is one of the best (free) way to support your favorite streamers and help them to grow up their channel. When you're hosting a channel, Twitch gives more exposure to the channel and it's a huge help to catch new people. More people host the channel, more exposure the channel earn.

Note: it seems auto hosting doesn't work anymore since Twitch updated some things a few months ago. So the goal of this guide is to explain you how to easily (manually) host a channel.

  • How to host a channel on a computer?
  • How to host a channel on a mobile device?

How to host a channel on a computer?

2 ways:

  • Type in your own chat /host ChannelName
  • The easiest way: Add a host button at the right of the sub/gift sub button

Type in your own chat /host ChannelName

Click on your avatar then on Creator Dashboard.

You should be in your Streamer Manager. At the right you have your own chat. Type /host ChannelName. Replace ChannelName by name of the channel so per example /host AlexKakoMusic.

Add a host button at the right of the sub/gift sub button

There is an easier way to host a channel on a computer. The BTTV extension. This extension add a lot of features to your Twitch. Focus on the HOST BUTTON feature.

First you have to install the BTTV extension. Click here and download it.

Then go to the BTTV settings. In whatever chat click on the gear icon then on BetterTTV settings.

Click on the channel settings icon, scroll down a bit and enable the HOST BUTTON feature.

Now you have a HOST BUTTON everywhere and the only thing you have to do to host a channel is to click on it.

How to host a channel on a mobile?

You can also host from anywhere using the iOS and Android mobile apps. To enable Host Mode on the go, tap the video player, then the Share button, then Host Channel.
